git remote add origin [URL] ربط المستودع المحلي بالمستودع البعيد لأول مرة git branch -M main تسمية الفرع الرئيسي (main) git push -u origin main رفع الفرع لأول مرة إلى المستودع البعيد مع تعيينه كافتراضي git push رفع التغييرات إلى المستودع البعيد (بعد ذلك) git pull جلب آخر التحديثات من المستودع البعيد git fetch جلب التحديثات بدون دمجها تلقائياً
